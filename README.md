# 🥤 음료 배합비 시뮬레이터 v3

Streamlit 기반 음료개발 통합 시뮬레이터 + Gemini AI + 나노바나나

## 실행
```bash
pip install streamlit pandas openpyxl
streamlit run app.py
```

## v3 주요 개선

| 기능 | 설명 |
|------|------|
| 🧪 산미료 pKa 보정 | 31종 pH·산도 숫자값 교체 (구연산 환산 표준) |
| 📐 실시간 pH 추정 | H⁺ 농도 가중합산 모델 |
| 📊 품질 예상치 테이블 | Brix/산도/pH/감미도/원가 ↔ 규격범위 판정 |
| 📋 좌측 표준배합비 | 참고표시 + 자동채움 버튼 |
| 🧠 원료명 유추 | 이름 규칙 → Brix/pH/산도 자동추정 |
| 🤖 Gemini AI 배합비 | 20년차 연구원 페르소나 |
| 🎨 나노바나나 이미지 | 제품사진 자동생성 |
| 📥 엑셀+HTML 출력 | 배합표+원가계산서 |
| 💾 값 보존 | 페이지 이동시 유지 |
