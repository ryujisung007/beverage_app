# 🥤 음료개발 데이터베이스 v3 — Streamlit 앱

## 실행 방법

### 1. 파일 준비
아래 3개 파일을 **같은 폴더**에 넣으세요:
```
📁 음료개발앱/
  ├── app.py                        ← Streamlit 앱
  ├── requirements.txt              ← 의존 패키지
  └── 음료개발_데이터베이스_v3.xlsx   ← 데이터 파일
```

### 2. 패키지 설치
```bash
pip install -r requirements.txt
```

### 3. 앱 실행
```bash
streamlit run app.py
```
브라우저에서 `http://localhost:8501` 자동 열림

## 📂 앱 구성 (10개 페이지)

| 페이지 | 내용 |
|--------|------|
| 🏠 대시보드 | DB 현황 요약, 원료/제품 분포 차트 |
| 🧪 배합시뮬레이터 | 음료유형+맛 선택 → 가이드배합비(AI/사례) 참조 → 배합비 입력 → 규격판정 |
| 💰 원가계산서 | 시뮬레이터 자동연동, 원재료+포장+제조 3단계 원가 |
| 📋 음료유형분류 | 17개 음료유형 정의/특성/대표제품 |
| 🏪 시장제품DB | 322건 시장제품 배합/규격 (시럽/잼/음료/통조림 포함) |
| 🧬 원료DB | 175종 원료 SPEC (Brix/pH/산도/감미도/단가) |
| 📏 음료규격기준 | 17개 유형별 당도/pH/산도/과즙함량 규격범위 |
| 🏭 HACCP | 표준제조공정 + CCP 관리기준 |
| 📊 자재SPEC참조 | 당류22종+감미료12종+안정제/부자재 참조표 |
| 🍎 과일Brix참조 | 60종 과일 한국/FDA 기준 Brix |
| 📖 가이드배합비DB | 10개 유형×맛 조합별 AI추천+실제사례 배합비 |
